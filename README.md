# Divine Palm Oracle 🌌

An immersive, AI-powered spiritual experience that reveals your destiny through ancient wisdom and palmistry.

## Features
- **Mystical Interface**: Immersive dark theme with floating particles and glowing auras.
- **Divine AI Oracle**: A god-like guide that speaks and listens to you.
- **Palmistry**: Upload images of your palms for AI-powered analysis.
- **Voice & Text**: Interact using your voice (Web Speech API) or traditional text input.
- **Multi-language**: Support for English, Malayalam, Hindi, and Arabic.
- **Ambient Sound**: Continuous mystical background music for deep immersion.

## Tech Stack
- **React (Vite)**
- **Tailwind CSS**
- **Motion** (Animations)
- **Gemini AI** (Destiny Analysis)
- **Web Speech API** (Voice Interaction)

## Getting Started

1. **Environment Variables**:
   Ensure `GEMINI_API_KEY` is set in your environment.

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```

## Deployment
This app is ready for deployment on **Netlify**, **Vercel**, or **Cloud Run**.
The `netlify.toml` is included for seamless Netlify deployment.
